# 😂 Joke Chat Bot

A simple chat-style web app that tells jokes using Python and Flask.

🔗 Live: https://jokes-7oqo.onrender.com/

---

## Features
- Chat-style interface 💬
- Random jokes 😂
- Sarcasm mode 😏
- Emoji responses 🎉

---

## Run locally

```bash
git clone https://github.com/DevBeginner7/Jokes.git
pip install -r requirements.txt
python app.py
```

## Tech
  - Flask
  - pyjokes
